## Usage

### NPM Script (Recommended)

```bash
npm run dev
```
Then open your browser on `http://localhost:40000/`.

### Hexo Command

Create New Post:

```
hexo n [postName]
```

Clean the cache:
```
hexo clean
```

Generate Pages:
```
hexo g
```

View Locally:
```
hexo s
```

Deploy
```
hexo d
```
